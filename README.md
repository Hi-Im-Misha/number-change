# number-change
Данный код меняет номера телефона начинающиеся с +38072, 38072, 072 на +7959

## Установка
Для установки нужно скачать папку index 
Добавить файл с номерами (Контакты.vcf) в корневую папку
Запустить файл index.exe

Вот пример где можно найти файл с номерами на Android
https://androidnik.ru/gde-v-android-xranyatsya-kontakty-i-kak-ix-dostat/
